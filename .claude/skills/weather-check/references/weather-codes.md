# 天气代码对照表

## 天气图标映射
| 天气状况 | 图标 | 建议 |
|---------|------|------|
| 晴 | ☀️ | 注意防晒 |
| 多云 | ⛅ | 正常活动 |
| 阴 | ☁️ | 正常活动 |
| 小雨 | 🌦️ | 带伞 |
| 中雨 | 🌧️ | 带伞，注意防水 |
| 大雨 | ⛈️ | 尽量避免外出 |
| 雷阵雨 | ⚡ | 注意防雷 |
| 雪 | ❄️ | 注意防滑 |
| 雾 | 🌫️ | 开车慢行 |
| 台风 | 🌀 | 避免外出 |

## 风力等级说明
| 等级 | 描述 | 影响 |
|------|------|------|
| 0-2级 | 微风 | 无影响 |
| 3-4级 | 和风 | 可以放风筝 |
| 5-6级 | 强风 | 打伞困难 |
| 7级以上 | 大风 | 避免户外活动 |

## 体感温度修正
- 湿度 > 70%：体感温度 = 实际温度 + 2°C
- 风力 > 3级：体感温度 = 实际温度 - 3°C
- 阴天：体感温度 = 实际温度 - 1°C
